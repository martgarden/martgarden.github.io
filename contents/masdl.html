MaSDL

Repozitář:Repository: https://bitbucket.org/mazahr/masdl/src/master/

Objektově orientovaný C++ wrapper na SDL z dob gymnázia. Nebyl nikdy úplně dokončen, ale úspěšně jsem ho použil například v programovací soutěži.

Jedoduché funkce jako vykreslení obdélníku (obdélník je objekt, který se sám vykresluje a může například měnit pozici).

Zpracování eventů pomocí dědění virtuální třídy (ala Java Interfaces).

Složitější funkce jako vypisování textu (wrapper pro SDL_TTF) nebo práce s pixelovými animacemi (soubor "sprite.cpp").

Object oriented C++ wrapper for SDL from my high school years. It has never been completely finished but I have used it sucessfully in some of my participations in programming competitions.

Simple functions like drawing rectangles (rectangle is an object capable of drawing it self and can for instance change position).

Event handling via class inheritance from virtual class (ala Java Interfaces).

More complicated functions like text rendering (wrapping SDL_TTF) or working with pixel animations (file "sprite.cpp").
